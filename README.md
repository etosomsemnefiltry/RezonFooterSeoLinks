# Rezon Footer SEO Category Links

Плагин для Shopware 6, который добавляет в футер сайта блоки с категориями для SEO-оптимизации.

## Возможности

- 5 настраиваемых блоков категорий
- До 5 категорий в каждом блоке
- Автоматическое формирование заголовков из breadcrumbs
- Адаптивная сетка (4-5 блоков на десктопе, меньше на планшете/мобильном)
- Настройки в админ-панели Shopware

## Установка

1. Скопируйте плагин в директорию `custom/plugins/` вашего Shopware проекта

2. Обновите список плагинов:
```bash
bin/console plugin:refresh
```

3. Установите и активируйте плагин:
```bash
bin/console plugin:install --activate RezonFooterSeoCategoryLinks
```

4. Очистите кэш:
```bash
bin/console cache:clear
```

## Компиляция

После установки плагина необходимо скомпилировать административную панель и storefront:

### Компиляция административной панели:
```bash
bin/console bundle:dump
bin/console administration:build
```

### Компиляция Storefront:
```bash
bin/console theme:compile
```

Или используйте скрипт компиляции вашей темы.

## Настройка

1. Перейдите в **Настройки → Система → Плагины**
2. Найдите плагин **Rezon Footer SEO Category Links**
3. Нажмите на **Настройки** (или перейдите в **Настройки → Система → Footer SEO Kategorien Links**)
4. Для каждого из 5 блоков выберите до 5 категорий через мультиселект
5. Сохраните настройки

```
    ├── views/

    └── snippet/                             # Переводы
```

## Как это работает

1. **Настройки**: Администратор выбирает категории для каждого блока в настройках плагина
2. **Данные**: При загрузке страницы `FooterSubscriber` получает выбранные категории из SystemConfig
3. **Заголовки**: Заголовки блоков формируются автоматически из breadcrumbs категорий (главная категория - название категории)
4. **Вывод**: Блоки отображаются в футере через переопределение шаблона `footer.html.twig`

## Адаптивность

- **Десктоп (>1200px)**: 5 блоков в строку
- **Планшет (992px-1200px)**: 4 блока в строку
- **Планшет (768px-992px)**: 3 блока в строку
- **Мобильный (576px-768px)**: 2 блока в строку
- **Мобильный (<576px)**: 1 блок в строку (столбик)

## Требования

- Shopware 6.7.0 или выше
